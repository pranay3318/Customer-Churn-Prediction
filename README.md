# Customer Churn Prediction.

A machine learning project for predicting customer churn using predictive analytics and data science techniques.

## 📋 Overview

This project analyzes customer behavior and builds a predictive model to identify customers who are likely to churn (discontinue their relationship with the company). By understanding churn patterns, businesses can implement targeted retention strategies.

## 🎯 Objectives

- Analyze customer data to identify churn patterns
- Build and train a machine learning model to predict customer churn
- Evaluate model performance using relevant metrics
- Provide insights into key factors contributing to customer churn

## 📊 Project Structure

```
Customer-Churn-Prediction/
├── customer_churn.ipynb    # Main Jupyter Notebook with analysis and modeling
└── README.md               # Project documentation
```

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- Required Python libraries (see Technologies Used section)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/pranay3318/Customer-Churn-Prediction.git
cd Customer-Churn-Prediction
```

2. Install required packages:
```bash
pip install pandas numpy scikit-learn matplotlib seaborn jupyter
```

3. Open the Jupyter Notebook:
```bash
jupyter notebook customer_churn.ipynb
```

## 📝 Usage

1. Open `customer_churn.ipynb` in Jupyter Notebook
2. Run the cells sequentially to:
   - Load and explore the data
   - Perform data preprocessing
   - Train the prediction model
   - Evaluate model performance
   - Generate predictions
